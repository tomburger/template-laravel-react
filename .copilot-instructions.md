# Copilot Instructions - Laravel + React Template

## Project Overview
This repository uses a single app root in `src`:
- Backend: Laravel API
- Frontend: React TypeScript app integrated with Vite

## Source Of Truth For Paths

- Laravel root: `src`
- React source: `src/frontend`
- Vite config: `src/vite.config.ts`
- Built frontend assets: `src/public/build`
- Start script (Windows): `start.cmd`

Do not assume `backend/` or `frontend/` folders at repository root.

## Technology Stack

### Backend
- Laravel (latest stable in this repository)
- REST JSON APIs
- Scribe for API documentation generation
- Eloquent ORM

### Frontend
- React 18+
- TypeScript
- Vite
- Bootstrap
- Axios
- Orval-generated endpoint clients

## Coding Standards

### PHP / Laravel
- Follow PSR-12
- Controllers and models use standard Laravel naming
- Keep validation and business rules on backend
- Return clear status codes and error messages

### TypeScript / React
- Functional components + hooks
- Handle loading and error states for API calls
- Keep components reusable and focused
- Match existing file naming conventions in `src/frontend`

## API Conventions

- Routes are under `/api/*`
- Use RESTful methods (GET, POST, PUT/PATCH, DELETE)
- Use JSON for requests/responses
- Keep response shapes consistent per endpoint family

## Authentication

- Use token-based auth as implemented in this repository
- Frontend includes `Authorization: Bearer {token}` when required

## Development Workflow

### Backend Tasks
- Run artisan commands from `src`
- Update migrations and API docs when endpoint contracts change

### Frontend Tasks
- Run npm scripts from `src`
- Use Vite live watching for local validation
- Do not run `npm run build` unless the user explicitly asks for a production build check

## Generation Guidelines

1. Keep backend logic out of UI components
2. Reuse existing endpoint/client patterns before adding new abstractions
3. Preserve current API contracts unless asked to change them
4. Add tests when touching behavior that is already covered by tests
5. Do not introduce Tailwind unless explicitly requested

## Common Commands

From repository root:

```bash
.\start.cmd
```

From `src`:

```bash
composer install
npm install
php artisan serve
npm run dev
```
